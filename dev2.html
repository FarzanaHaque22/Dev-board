Question-1: What are the different ways to select an element in the DOM?

Selecting elements in the DOM (Document Object Model) is a fundamental task in web development: getElementById, getElementsByClass,getElementsByTagName,querySelector,querySelectorAll,getElementsByName

Question-2: What is the difference between innerHTML, innerText, and textContent ?

When you use the innerHTML property, it reads both the HTML markup and the text content of the element.This property focuses on the rendered text content. When you use innerText to read the content of an element, it returns the text as it appears on screen.The textContent property also ignores all HTML tags and returns only the text.

Question-3: What is event delegation in the DOM?

Event delegation is a powerful technique in JavaScript that allows you to manage events efficiently, especially when dealing with a large number of elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a method of event propagation in the HTML DOM API. It is a process that starts with the element that triggered the event and then bubbles up to the containing elements in the hierarchy

Question-5: How do you create, add, and remove elements using JavaScript?

To remove an element, you can use the removeChild method or the remove method.To add an element to the DOM, you can use the appendChild or insertBefore methods.
